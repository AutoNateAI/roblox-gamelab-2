AI Software Architect Path

Students crash-land into Screeps, a living programming world where every worker is controlled by code. Over 3 weeks, they set up a local development environment, learn Git, collaborate with AI, build a working colony bot, and finish with an intra-cohort battle day.

Enrolling Now3 weeks · Tuesdays and Thursdays, 6:30 PM-8:30 PM Eastern
Start
Program week opens Monday, August 3, 2026. Live sessions begin Tuesday, August 4, 2026.
Learning outcomes
Think systematically; describe software behavior in plain language; use AI coding tools responsibly; read and improve generated JavaScript; use Git to store and share code; debug a running system from real feedback.
Project
A local Screeps bot that can harvest energy, spawn workers, upgrade a controller, organize roles, store code in Git, and compete in a final private-server battle day.
Portfolio artifact
A student-owned Git repository with Screeps bot code, learning notes, screenshots, and a final battle-day reflection.

Inside the 3 weeks

01

Session 1: Welcome to the World

Tuesday, August 4, 2026

Understand what Screeps is, connect to the local/private server, create the first room, place a spawn, and run the first working creep command.

Live: Instructor-led setup: Screeps client, local repo, Discord orientation, first room selection, first spawn, first worker.Homework: Capture one screenshot of your room and write a short note explaining what a spawn, source, creep, and controller do.
02

Session 2: The First System Loop

Thursday, August 6, 2026

Build the first harvest-and-return loop and connect game behavior to real-world systems like supply chains and operations.

Live: Students implement a basic harvester loop, observe errors, and practice describing failures clearly before asking AI for help.Homework: Let the bot run, log 3 things it does well, and log 3 things that need improvement.
03

Session 3: Roles, Memory, and Git

Tuesday, August 11, 2026

Introduce role-based architecture, persistent Memory, and Git commits as a way to preserve working progress.

Live: Create harvester/upgrader/builder roles, initialize or update the program repo, make the first meaningful commits.Homework: Push code to Git and write a plain-English explanation of each role.
04

Session 4: Debugging a Living Colony

Thursday, August 13, 2026

Learn to debug from game state: idle spawns, missing energy, stuck creeps, broken memory, and unclear prompts.

Live: Students diagnose real bot failures, turn observations into AI prompts, review suggested code, and apply only what they understand.Homework: Open one issue in the repo describing a bug or improvement in the bot.
05

Session 5: Battle Prep and Strategy

Tuesday, August 18, 2026

Connect economy, movement, defense, and simple combat strategy to the final battle-day server.

Live: Prepare a battle-ready branch, discuss simple attack/defense loops, and practice reading opponent behavior as a system.Homework: Submit the battle branch and write a 5-sentence strategy note.
06

Session 6: Battle Day

Thursday, August 20, 2026

Compete in an intra-cohort tournament, observe what strategies hold up, and reflect on systems decisions under pressure.

Live: Private-server PvP bracket. Students run their bots, watch matches, discuss failures, and document what they would improve next.Homework: Submit final repo link, screenshots, and battle-day reflection.
