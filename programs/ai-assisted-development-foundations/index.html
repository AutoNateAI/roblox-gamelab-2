AI-Assisted Development Foundations

Build a working single-room Screeps economy bot (spawn, harvest, build, upgrade) using an AI coding agent (Codex/Claude Code/Gemini CLI) before understanding syntax. Screeps is a real JavaScript programming game — you write code that controls persistent units in a live world — so 'working' means the bot actually runs and survives.

Inside the 3 weeks

01

Session 1: Meet Your AI Coding Agent

Set up a Screeps account and room; use an AI coding agent to write a first working creep spawn and harvest script.

Live: Instructor demos prompting an AI agent to spawn a creep and harvest energy; students mirror it live in their own room.Homework: Create a free Screeps account and claim a starting room before next session.
02

Session 2: Describing a Full System

Describe a full system (spawn, harvest, upgrade, build) in plain language and have the AI implement it end-to-end.

Live: Students write a plain-English spec for a 3-role economy (harvester/upgrader/builder), then prompt their AI agent to build it.Homework: Let the bot run overnight; screenshot the room state the next morning.
03

Session 3: Iterating on Real Feedback

Practice iterative prompting: adjust spawn logic based on real room needs.

Live: Review each bot's overnight behavior; prompt the AI to fix one real issue observed.Homework: Log 3 bugs or weird behaviors observed in the overnight run.
04

Session 4: Basic State with Memory

Introduce Screeps' Memory object; assign each creep a role/state/target using AI-assisted code.

Live: Add per-creep Memory (role, state, target) with AI help; observe how state persists across ticks.Homework: Sketch what states each creep role can be in.
05

Session 5: Watch It Run, Then Describe It

Practice describing bugs in natural language instead of pasting raw errors; watch how much better AI fixes are when the description is clear.

Live: Each student narrates a failure from their bot's run to the class, then converts it into a prompt for their AI agent.Homework: Write a one-paragraph bug report in plain English for the AI, no code pasted.
06

Session 6: Project Checkpoint - Explorer Badge

Ship a stable single-room economy bot; explain what it does in plain terms.

Live: Demo day: each student shows their working bot surviving a full day, and explains it to a partner.Homework: Record a 2-minute walkthrough video of the bot for your portfolio.
