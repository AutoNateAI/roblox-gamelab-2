# Roblox GameLab quest skills

These skills turn opportunity research into short Roblox quests that can be
published as AutoNateAI GameLab Season 1 content.

1. [Season scout](season-scout.md) - choose a trend, program, or partner angle
   worth building.
2. [Quest composer](quest-composer.md) - turn the angle into acts, zones,
   missions, power-ups, obstacles, and a capstone.
3. [Marketplace publisher](portal-publisher.md) - publish the quest brief to the
   AutoNateAI GameLab marketplace/API namespace.

Use `data/season-1/quests.json` as the source seed for Season 1 until Firestore
becomes the primary authoring database.
